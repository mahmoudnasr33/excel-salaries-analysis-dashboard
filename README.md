# Salaries Analysis Dashboard (Interactive Dashboard using MS Excel)
## Project Objective
Build an interactive Excel dashboard to analyze salary distribution and trends across departments, job roles, locations, education levels, and experience.

## Dataset Used
- [Download Dataset](https://github.com/mahmoudnasr33/excel-salaries-analysis-dashboard/blob/main/Dataset.xlsx)

## Questions (KPIs)
- What is the total salary?
- What is the average salary by department?
- What is the average salary by location?
- Which job roles have the highest average salaries?
- How does salary change with experience?
- What is the impact of education level on salary?
- Gender distribution across employees.

## Process
- Cleaned and validated dataset to handle missing or inconsistent values.
- Standardized salary and experience formats.
- Created pivot tables for department, location, education, and job analysis.
- Built dynamic charts using pivot charts.
- Applied slicers for interactive filtering.
- Designed a professional dashboard layout for clear visualization.
